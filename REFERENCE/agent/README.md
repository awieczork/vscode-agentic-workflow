# Agent Reference

Quick reference for creating and validating `.agent.md` files.

---

## Files

| File | Purpose | Use When |
|------|---------|----------|
| [PATTERNS.md](PATTERNS.md) | When/why/rules | Understanding agent design decisions |
| [TEMPLATE.md](TEMPLATE.md) | Format/structure | Creating new agents |
| [CHECKLIST.md](CHECKLIST.md) | Validation | Before committing agent files |
| [TAGS-AGENT.md](TAGS-AGENT.md) | Tag vocabulary | Selecting appropriate tags |

---

## Quick Start

1. **Decide** — Read [PATTERNS.md](PATTERNS.md) to confirm you need an agent (not instruction/skill/prompt)
2. **Create** — Copy minimal template from [TEMPLATE.md](TEMPLATE.md)
3. **Fill** — Use [TAGS-AGENT.md](TAGS-AGENT.md) for valid tags and examples
4. **Validate** — Check against [CHECKLIST.md](CHECKLIST.md) before committing
